# 🌍 PLANET TURBO

Audio-reactive flight + ring game / VJ toy. Single self-contained HTML file (Three.js via CDN, no build step).

**Play:** open `index.html` or visit the GitHub Pages link.

## Controls
- **WASD** — steer (bank-to-turn) + pitch
- **SPACE** — boost (camera pulls in close)
- **E** — autopilot (auto-fly through the ring path)
- **START MIC** — audio-reactive mode (jungle/club VJ): the floating objects pump to the music

Fly the cute 1-seat plane piloted by the Planet Turbo globe mascot, thread the golden rings (the character does a random flip/hop on each pass), and fill the boost gauge. The jungle-party-crew mascot © Planet Turbo.
